# Selenium Java Automation

## Overview
This repository contains Selenium WebDriver automation examples developed using Java, TestNG, and Maven.

## Technologies
- Java
- Selenium WebDriver
- TestNG
- Maven

## Features
- Browser Launch
- Login Automation
- Search Functionality
- Form Submission
- Dropdown Handling
- File Upload
- Assertions

## Purpose
This is a personal learning project created to practice Selenium automation testing and improve automation skills.
